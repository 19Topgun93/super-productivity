The global link lists

The global link list right under the navigation offers to possibility to add links to your project. This can be done by clicking on the '+'-button in the top left, via drag and drop or via pasting a previously copied link while the application has focus.

Links can be edited and deleted via the buttons that appear when hovering a link.

Files can also be linked as can custom commands (be careful with this!).

Hover the example to see what everything does.
