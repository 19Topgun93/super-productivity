Git integration

Here you can configure to list open github issues for a specific repository in the task creation panel on the daily planning view. They will be listed as suggestion and will provide a link to issue as well as several other information about it.

In addition you can automatically add and sync ALL open issues to your task backlog.

Pull requests will also be treated as a task, you can add a custom prefix to distinguish them easier from other issues.

You can also configure the path to your projects .git directory to show you a summary of commits at the end of the day.
