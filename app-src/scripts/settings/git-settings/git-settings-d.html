Git integration

Here you can configure to list open github issues for a specific repository inside the task creation panel. They will be listed as suggestion and will provide a link to issue as well as several other information about it.

Pull requests will also be shown as task suggestion, you can add a custom prefix to distinguish them easier from other issues.

You can also configure the path to your projects .git directory to show you a summary of commits at the end of the day.
